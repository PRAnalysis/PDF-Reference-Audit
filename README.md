# PDF Reference Audit

The PDF Reference Audit is a systematic method for analyzing and validating the references cited within a PDF document. It is designed to assess the integrity, traceability, and reliability of cited sources, particularly in research, policy, technical, or investigative contexts.

The audit focuses on what a document claims to rely on, rather than the claims themselves.


# Purpose

The reference audit is used to:

Extract bibliographic references from PDFs

Identify unique citations (e.g., DOIs, authors, titles)

Detect retracted, missing, unverifiable, or malformed references

Assess citation quality and dependency patterns

Support transparency, replication, and accountability

This is especially useful for:

Research validation

Policy and regulatory review

Meta-research and evidence audits

Investigative or watchdog work
